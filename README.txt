WINDOWS
1)Aprire cartella progetto con terminale
2)python -m venv venv (comando per creare virtual environment)
3)venv\Scripts\activate.bat (comando per attivare il venv)
4)pip install -r requirements.txt (comando per installare i packages necessari all'interno del venv)
5)python main.py num_cities map_size (num_cities e map_size sono parametri interi)
6)deactivate (comando per uscire dal virtual environment)

Unix/macOS
1)Aprire cartella progetto con terminale
2)python -m venv venv (comando per creare virtual environment)
3)source venv\bin\activate (comando per attivare il venv)
4)pip install -r requirements.txt (comando per installare i packages necessari all'interno del venv)
5)python main.py num_cities map_size (num_cities e map_size sono parametri interi)
6)deactivate (comando per uscire dal virtual environment)
